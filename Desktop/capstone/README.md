# 북뮤직 (BookMusic) 🎵📚

**책과 음악을 연결하는 웹 플랫폼**

책을 읽고 느낀 감정을 음악으로 표현하고 공유할 수 있는 플랫폼입니다. 책의 인상 깊은 구절을 바탕으로 음악 플레이리스트를 생성하고, 다른 사용자들과 공유할 수 있습니다.

## 🚀 빠른 시작

### 1. Node.js 설치
[Node.js 공식 홈페이지](https://nodejs.org/)에서 LTS 버전을 다운로드하여 설치하세요.

### 2. 프로젝트 실행
```bash
# 의존성 설치
npm install

# 서버 실행
npm start
```

### 3. 웹사이트 접속
브라우저에서 **http://localhost:3000** 에 접속하여 확인하세요.

## 🎯 핵심 기능

| 기능 | 설명 |
|------|------|
| **🎵 음악 생성** | 책 정보와 인상 깊은 구절을 입력하여 AI 음악 플레이리스트 생성 |
| **📋 게시판** | 사용자들의 플레이리스트를 앨범형 레이아웃으로 공유 및 탐색 |
| **📚 나의 책장** | 개인 플레이리스트를 관리하고 새로운 플레이리스트 생성 |
| **👤 마이페이지** | 프로필 관리 및 개인 정보 수정 |

## 🎨 테스트 계정

- **아이디**: `user`
- **비밀번호**: `password`

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Node.js, Express.js
- **Storage**: LocalStorage (프로토타입용)

---

> **참고**: 현재 프로토타입 버전으로 음악 생성 기능은 시뮬레이션되며, 데이터는 로컬 스토리지에 저장됩니다.
